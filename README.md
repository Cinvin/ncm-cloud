# 网易云音乐云盘管理
## 功能
* 我的云盘文件（删除云盘文件，设置关联的歌曲）
* 本地上传文件
* 咪咕音源上传  (按歌手或按专辑批量上传)
  
### [Releases](https://github.com/Cinvin/ncm-cloud/releases) 页面可下载安装包
