## 部署开发环境

```shell
$ git clone git@github.com:Cinvin/ncm-cloud.git

$ cd ncm-cloud

$ npm install
# 调试
$ npm run dev
# 打包
$ npm run build
```
在macOS环境下开发，暂未在Windows调试打包过
